# El Rincón de mi Nostalgia
Como parte del módulo I del curso de desarrollo FronEnd, y haciendo uso, principalmente, de HTML y cosas básicas de CSS, es que nace este pequeño proyecto que combina dos de mis pasiones: la escritura y el desarrollo web.  
Motivada por mi personalidad nostálgica y meláncolica, tomo la decisión de crear un pequeño rincón que alberga, de manera muy general, mi caracter repleto de melancolía.

## Tecnologías empleadas: HTML5 y CSS3
La estructura general se sustenta por HTML semántico, mientras que los estilos fueron añadidos con CSS, no obstante, no se añadió un archivo externo, sino que se hizo uso de estilos lineales, esto es, implementados en el atributo de la etiqueta sobre la cual se deseaba añadir algún detalle estético.

## Propósito: Demostrar que se han adquirido los conocimientos básicos de HTML
En esencia, el objetivo de este minisitio web es evidenciar que a lo largo de las primeras semanas del curso, se logró aprender lo básico de HTML5, así como, el actual uso del denominado HTML semántico que, en lugar de usar etiquetas div, las sustituye según su objetivo.  
Lo anterior quiere decir que, si se crea una pie de página, en lugar de encerrarlo en un div, se crea con la etiqueta footer.

## Descripción del sitio web
El sitio web "El Rincón de mi Nostalgia" se compone de 5 páginas:  
- Página principal
- Fotografías
- Vídeos
- Cartas
- Reflexiones

Cada página tiene en común los estilos del `body`, la barra de navegación y el pie de página.  

La estética del sitio se basa en el diseño del antiguo Sistema Operativo Windows95, un SO que no tuve la dicha de probar pero que cuyos recuerdos en internet me hacen sentir atrapada en el tiempo, en un mundo altamente pixelado.  
No me interesa si fue un SO que actualmente es incapaz de compertir con los SO más modernos como Windows11, simplemente adoro ese background menta oscuro, esas ventanas no minimalistas.  
Extraño cuando la vida se sentía, se veía. Cuando el maximalismo se anteponía a lo simple y vago.  
Esa, fue la inspiración para la estética de este ministio web.
